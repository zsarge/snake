# snake

Inspired by [this video](https://youtu.be/TOpBcfbAgPg)

This file uses the snake_case naming convention, for obvious reasons.

## in the project directory, build with:

```
mkdir build
```

```
cd build
```

```
cmake ..
```

```
make
```

```
./snake.out
```
